因为我是自己上传的 github ，然后打不开了，所以查了一下 ip 和相关文章。网上文章都是在 C:\Windows\System32\drivers\etc\hosts 文件中复制固定内容，不同网段的用户查到的 IP 可能不一样，所以需要自己去查，而不是简单的copy。
